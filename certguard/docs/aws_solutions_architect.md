# AWS Certified Solutions Architect - Associate (SAA-C03) Certification Guide

The AWS Certified Solutions Architect - Associate certification validates your capability to design secure, resilient, high-performing, and cost-optimized architectures on AWS.

## Core Required Skills
- **aws**: Fundamental understanding of AWS cloud infrastructure.
- **ec2**: Designing compute solutions, choosing instance types, configuring Auto Scaling, and load balancing (ALB/NLB).
- **s3**: Designing storage solutions, lifecycle policies, encryption, versioning, and bucket policies.
- **rds**: Configuring relational databases, Multi-AZ deployments, read replicas, and backup policies.
- **vpc**: Designing network architectures, subnets, routing tables, NAT Gateways, Security Groups, and NACLs.
- **iam**: Configuring security policies, roles, user permissions, and multi-factor authentication.
- **serverless**: Building serverless architectures using Lambda, API Gateway, DynamoDB, and SQS/SNS.

## Typical Study Plan
- **AWS Cloud Practitioner (CLF-C02)**: 10 hours (optional, for cloud beginners).
- **Compute and Storage (EC2, S3)**: 20 hours of hands-on practice.
- **Networking and Security (VPC, IAM)**: 25 hours of network configuration and security audits.
- **Databases and Serverless**: 20 hours of DynamoDB, RDS, and Lambda deployment.
- **Architecture Design Patterns**: 25 hours of well-architected framework review and practice tests.

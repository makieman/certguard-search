# Microsoft Certified: Azure Solutions Architect Expert (AZ-305) Guide

The Azure Solutions Architect Expert certification demonstrates your ability to design cloud and hybrid solutions that run on Microsoft Azure, covering compute, network, storage, monitoring, and security.

## Core Required Skills
- **identity_governance_design**: Designing organizational hierarchy, Entra ID integration, directory synchronization, and governance rules.
- **data_storage_design**: Architecting database schema storage solutions, sizing, caching, and database lifecycle management.
- **business_continuity_design**: Designing backup and recovery strategies, site recovery, high availability, and disaster recovery.
- **infrastructure_design**: Architecting network topologies, virtual networks, ExpressRoute, and Azure load balancing strategies.

## Study Timeline & Budget
- **Total Recommended Study**: 60-80 hours.
- **Weekly Schedule**: 8-10 hours over 8 weeks.

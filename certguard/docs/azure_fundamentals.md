# Microsoft Certified: Azure Fundamentals (AZ-900) Guide

The Azure Fundamentals certification verifies foundational knowledge of cloud concepts and core Azure services, management, governance, security, and privacy.

## Core Required Skills
- **cloud_concepts**: Understanding high availability, scalability, elasticity, agility, fault tolerance, and disaster recovery.
- **azure_architecture_services**: Working with Azure subscriptions, resource groups, regions, availability zones, virtual machines, and app services.
- **azure_management_governance**: Configuring Azure portal, Azure PowerShell, CLI, Azure Resource Manager (ARM), and budget alerts.
- **azure_security_compliance**: Configuring network security groups, firewall, Microsoft Defender for Cloud, and Microsoft Entra ID.

## Study Timeline & Budget
- **Total Recommended Study**: 15-20 hours.
- **Weekly Schedule**: 5 hours over 3-4 weeks.

# Python Certified Associate Programmer (PCAP) Certification Guide

The PCAP certification focuses on the object-oriented programming (OOP) approach to Python, checking the learner's ability to accomplish coding tasks related to core programming concepts.

## Core Required Skills
- **python**: In-depth knowledge of Python syntax, data types, control structures, and built-in functions.
- **oop**: Implementing classes, objects, inheritance, polymorphism, encapsulation, and method overriding in Python.
- **modules**: Importing and creating modules, using packages, and working with Python's standard library modules (os, sys, math, datetime).
- **exceptions**: Try-except blocks, raising exceptions, custom exception classes, and debugging code.
- **testing**: Writing unit tests using `unittest` or `pytest`, writing assertions, and mock testing.

## Typical Study Plan
- **Python Syntax & Basics**: 15 hours of basic scripting, functions, and list comprehensions.
- **Object-Oriented Programming (OOP)**: 25 hours designing class hierarchies, inheritance, and magic methods.
- **Standard Library & Modules**: 15 hours practicing file operations, os/sys calls, and date/time manipulation.
- **Exception Handling & Testing**: 15 hours of test-driven development (TDD), writing tests, and exception handling.

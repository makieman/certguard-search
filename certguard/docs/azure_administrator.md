# Microsoft Certified: Azure Administrator Associate (AZ-104) Guide

The Azure Administrator certification demonstrates your skills in implementing, managing, and monitoring an organization's Microsoft Azure environment.

## Core Required Skills
- **azure_identities_governance**: Managing Microsoft Entra users, groups, licenses, role-based access control (RBAC), and subscriptions.
- **azure_storage_implementation**: Creating and configuring storage accounts, blob storage, Azure Files, and storage security.
- **azure_compute_resources**: Configuring VMs, containers, Azure App Services, and Azure Kubernetes Service (AKS).
- **azure_virtual_networking**: Designing virtual networks, subnets, public/private IPs, network security groups, VPN gateway, and peering.
- **azure_resource_monitoring**: Configuring Log Analytics workspace, Azure Monitor alerts, and VM performance insights.

## Study Timeline & Budget
- **Total Recommended Study**: 60-80 hours.
- **Weekly Schedule**: 8-10 hours over 8 weeks.

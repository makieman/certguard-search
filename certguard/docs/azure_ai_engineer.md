# Microsoft Certified: Azure AI Engineer Associate (AI-102) Guide

The Azure AI Engineer Associate certification validates your skills in building, managing, and deploying AI solutions that leverage Azure Cognitive Services, Azure OpenAI, and machine learning.

## Core Required Skills
- **azure_ai_services**: Creating and configuring Azure AI Services resources, managing keys, endpoints, and security policies.
- **natural_language_processing**: Implementing text translation, language detection, custom question answering, and conversational language understanding.
- **computer_vision_solutions**: Creating image analysis pipelines, custom vision classifiers, optical character recognition (OCR), and video indexing.
- **generative_ai_integration**: Implementing Azure OpenAI models, Prompt Engineering, Retrieval-Augmented Generation (RAG), and content safety filters.

## Study Timeline & Budget
- **Total Recommended Study**: 50-60 hours.
- **Weekly Schedule**: 8 hours over 7 weeks.

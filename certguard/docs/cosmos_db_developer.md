# Microsoft Certified: Azure Cosmos DB Developer Specialty (DP-420) Guide

The Azure Cosmos DB Developer Specialty certification validates your ability to design and implement cloud-native applications using Microsoft Azure Cosmos DB.

## Core Required Skills
- **cosmosdb_data_modeling**: Designing JSON document structures, managing relational data modeling, and choosing embed vs reference models.
- **cosmosdb_partitioning**: Defining partition keys, avoiding hot partitions, and configuring physical/logical partitions.
- **cosmosdb_replication**: Implementing global replication, configuring multi-region writes, and selecting consistency levels.
- **cosmosdb_query_tuning**: Optimizing SQL queries, measuring Request Unit (RU) costs, and configuring index policies.
- **cosmosdb_sdk_integration**: Integrating .NET/Java/Python/Node.js SDKs, handling throttling errors, and writing stored procedures.

## Study Timeline & Budget
- **Total Recommended Study**: 40-50 hours.
- **Weekly Schedule**: 6-8 hours over 6 weeks.
